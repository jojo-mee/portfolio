Interactive Elements

In-text link

This is an in-text link.

Sample code:

<p>This is an <a class="intext_link" href="#">in-text link</a>.</p>

Button Link

Default Button

Sample code:

<a href="#" class="button">Default Button</a>

Submit/Reset Buttons

Sample code:

<form action="/">
  <button type="submit" value="Submit">Submit</button>
  <button type="reset" value="Reset">Reset</button>
</form>

Text Input Field with Label

Sample code:

<form action="/">
  <label>Name:</label> <input type="text" placeholder="Jo Zhou">
</form>

Radio Button with Label

Sample code:

<form action="/">
  <label class="form_text">Gender :</label>
  <label class="form_radio"><input type="radio" name="sex"> Male </label>
  <label class="form_radio"><input type="radio" name="sex"> Female </label>
</form>

Image as a Link

Click the image below to see detail.

Sample code:

<a class="imageLink" href="#">
  <img class="record" src = "img/record.jpg" alt = "photo"/>
</a>

Text Elements

Headings

heading1: for top title

heading2: for section title

heading3: for element title

Sample code:

<h1 class="section_heading">heading1</h1>
<h2 class="element_heading">heading2</h2>
<h3>heading3</h3>
